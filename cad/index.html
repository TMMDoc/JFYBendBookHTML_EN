CAD

CAD

Before you can do any sheet-metal processing (for press-brakes, panel benders, or laser cutters), you need to start with CAD data. When using JFY Bend, this CAD data can come from the following sources:

  • Import of a flat pattern with bend information from a GEO file. This drawing is already annotated with bend lines, and bending information, so it can be readily folded into a sheet metal 3D model.

  • Import of a flat pattern from a DXF file. These drawings may need some pre-processing before they can be used by the Bend or Cut modules:

    • You may need to specify a material and a thickness.

    • You may need to tag some of the lines in the drawing as bend lines and provide bend information (like the bending angle, direction and radius).

  • Import of a sheet metal model from various 3D formats (like IGS, STEP, SLDPRT, STEP etc). These sheet metal models can be unfolded to create sheet metal flats for laser cutting, or can be processed directly by the bending module.

  • Import of a 3D assembly model from various 3D formats (like STEP, SLDASM etc). The assembly can then be exploded by JFY Bend and each of the resultant components that are sheet metal models can be processed using the modules like Bend or Cut.

  • Creation of a sheet metal flat pattern from scratch, using the Sketch functions built into JFY Bend.
