Collisions with the Blank Holder

This collision is also an error during the release movement. The blank holder collides with the bending profile.

Currently, there is no option in {app name} to fix errors of this kind.

To solve this problem, you can change the following parameters on the HMI:

  • In FZ10: set "Parameter1" = 1 mm.

  • In FZ35 with "Parameter 4" and insert a value of 1 mm.

  • In FZ11: set "Parameter 2" = 1 mm.
