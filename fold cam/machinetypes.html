Machine Selection

In order to select a panel bender in {app name}, follow the below steps:

  1. Click Database icon from the command bar.

  2. Select Machines from the dropdown list.

  3. Choose the desired machine from the list.

Alternatively, if you know the machine ID you can directly use the search function.
