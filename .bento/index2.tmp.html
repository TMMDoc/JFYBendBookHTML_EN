Thank you for choosing JFY Bend.

JFY Bend is an Integrated Sheet-metal CAM application. Using JFY Bend, you can convert your sheet metal designs into CAM data.

JFY Bend can generate code for a large number and variety of sheet-metal machinery, including:

  • Press-brakes

  • Panel bending machines

JFY Bend leverages a unified environment, from which one can program all the different technology machines within the sheet metal shop floor.

JFY Bend can integrate with Praxis Plus system seamlessly for doing advanced resource planning, shop-floor management and large-volume batch processing.

An innovative and futuristic, yet easy to learn tool, for the classic sheet metal factory setup, providing optimal performance deliverance of productivity.
