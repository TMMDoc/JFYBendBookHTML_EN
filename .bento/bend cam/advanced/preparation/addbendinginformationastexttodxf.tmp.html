1. Open drawing.

  2. Create text with the following format: A-90 R1.5 K419.

    1. A: Value for the bending angle can be assigned here. Depending on what has been selected in the angle is an opening angle or a bending angle (external angle).

    2. R: Value for the bend radius can be assigned here.

    3. K: Value for the correction factor k can be assigned here. {appName} detects whether an ISO or DIN correction factor k has been assigned.

    4. RM: 1.0038: This type of text can be used to define a material. The name used can be one of the materials or one of the alternative names.

    5. THK: 2.5: This type of text can be used to indicate the thickness of the sheet in mm.

  3. The text must touch the line, which is to be converted to a bending line.

  4. Save the drawing as a DXF.

  5. Open the DXF in {appName}.

    1. {appName} converts the line with text into a bending line during import. The text is then no longer displayed.

    2. A polyline can also be used instead of a line in an individual linear segment.
