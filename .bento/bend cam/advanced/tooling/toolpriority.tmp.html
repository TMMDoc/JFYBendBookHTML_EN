A tool priority can be set in the tool management, in the list of selected tools. Whenever possible, use tools with higher priority for the calculation.

To specify a priority:

  • Press at the Tool Priority symbol to reduce the number of the tool priority (increases priority).

  • Press at the Tool Priority symbol to increase the number of the tool priority (decreases priority).
