There is a wide variety of optional ENW blank-holders, and ZBW blades that you may use with your panel bender. In addition, some of the components of the panel-bender such as the upper or lower blank-holder, and the folding blades may be swapped out with different ones.

Figure 1. Tooling overview

The Tool Inventory window in JFY Bend is used to configure these attachments and tools for the panel-bender.

To access the Tool Inventory window, open a part, and switch it to panel bending mode. An inventory button appears in the command bar, and clicking on this opens up the Tool Inventory window.

In order to view all tool types in the inventory, open {app name} in admin mode.

Editing ZBW Blades or ENW Tools

Use the Type selector to pick the ENW tools or ZBW blades that are available for JFY Bend to use. When editing these tools, clicking on a tool toggles between being available or unavailable. In addition, you can rearrange these tools in a diferent order (by clicking and dragging a tool to a different location). JFY Bend uses this ordering as the tool priority, and tries to use these tools in the specified order. For example, in the image above, JFY Bend will try to process a part using the ENW110-50 tool before it tries to use the ENW110-30 tool.

Editing Machine-Configuration

Some of these types (like the blank-holder, upper-blade, or lower-blade) do not allow multiple-selections. In a sense, these types represent the current configuration of the machine, and only one blank-holder may be mounted on the machine at one time. When the upper or lower blade is changed on the machine, for example, this section is used to inform JFY Bend of the change, so it can tool up parts and do collision checks appropriately.

Depending on your JFY Bend installation, it is likely that these sections (like blank-holder, upper-blade or lower-blade) may not be accessible for editing except by a Trumpf technician. All these settings are for the currently selected panel-bender (the one being used for the current part). If you have multiple panel benders, you can vary these settings individually for each one.

Importing New Tool Data

When you get new tools from Trumpf, these tools will be accompanied by tool-data files (tdata files), that are used to inform {app name} about the geometry and capabilities of these new tools. You can import these new tdata files into {app name} using the Add New button. This opens a file selector that lets you select one or more tdata files. These data are added to the tool catalogue, and can then be selected in the various pages of the Tool Inventory window.

To import special tool ordered from TRUMPF proceed as follows:

  1. Check that TRUMPF has provided a TData-Files.zip and unzip it.

  2. Open any program in {app name}.

  3. Navigate to Tool management.

  4. Click the Inventory icon from the command bar.

  5. Under Type, select the type of tool you want to create and click on New.

  1. Navigate to the tool you wish to add.

  2. Select the .tdata file and click Open button.

The tool is now displayed in your tool database.

Check that the quantity of the relevant size corresponds to the tool set you have received.
