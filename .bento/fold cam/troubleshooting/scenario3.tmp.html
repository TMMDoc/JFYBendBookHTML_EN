Two problems occur simultaneously here:

  1. The standard blank holders are too low, as they collide on the rear side during folding.

  2. The length is too short to be able to create a bend.

The recommended solution here is to use a different blank holder, i.e. an ENW tool.

  • Click on the blank holder, select Blank holder as ENW tool and select the ENW tools you want to use

  • Adjust the blanl holder length so that no collision is displayed.
